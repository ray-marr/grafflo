## Getting Started

Run the development server:

```bash
npm install
npm run dev
```

## About the app

Small demo app of an interactive price to earnings graph built with d3js

![Price to earnings graph](public/images/priceEarnings.png)

## Stack used

- NextJs 15
- Material UI
- D3
- SASS
- Typescript
